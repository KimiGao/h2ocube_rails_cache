# H2ocubeRailsCache

Just an redis cache.

## Installation

Add this line to your application's Gemfile:

    gem 'h2ocube_rails_cache'

Add this line to config/environments/production.rb:

    config.cache_store = :h2ocube_rails_cache

And then execute:

    $ bundle
